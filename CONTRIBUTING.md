Contributing

Thank you for your interest in contributing to this project. Please follow these guidelines:

- Fork the repository and create a feature branch named `feature/<short-description>`.
- Make small, focused commits with clear messages (imperative style).
- Run the simulator and tests locally before opening a PR.
- Open a Pull Request describing your change and reference any related issue.
- Keep secrets out of commits; use `.env` for keys and never commit it.

Reporting issues
- File issues with a clear title, steps to reproduce, and expected vs actual behavior.

License and code of conduct
- This project uses the MIT license. Please be respectful and professional in discussions.
